[Chapter 21] ContainerListener Chapter 21java.awt.event Reference ContainerListener Name ContainerListener Description Objects that implement the ContainerListener interface can receive ContainerEvent objects. Listeners must first register themselves with objects that produce events. When events occur, they are then automatically propagated to all registered listeners.
