[Chapter 21] FocusAdapter Chapter 21java.awt.event Reference FocusAdapter Name FocusAdapter Description The FocusAdapter class implements the methods of FocusListener with empty functions. It may be easier for you to extend FocusAdapter, overriding only those methods you are interested in, than to implement FocusListener and provide the empty functions yourself.
