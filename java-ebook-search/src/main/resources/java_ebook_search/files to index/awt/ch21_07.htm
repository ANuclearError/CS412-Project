[Chapter 21] ComponentListener Chapter 21java.awt.event Reference ComponentListener Name ComponentListener Description Objects that implement the ComponentListener interface can receive ComponentEvent objects. Listeners must first register themselves with objects that produce events. When events occur, they are then automatically propagated to all registered listeners.
