[Chapter 11] PipedWriter Chapter 11The java.io Package PipedWriter Name PipedWriter Synopsis Class Name: java.io.PipedWriter Superclass: java.io.Writer Immediate Subclasses: None Interfaces Implemented: None Availability: New as of JDK 1.1 Description The PipedWriter class represents half of a communication pipe; a PipedReader must be connected to a PipedWriter. When the two halves of a communication pipe are connected, data written to the PipedWriter can be read from the PipedReader. The communication pipe formed by a PipedWriter and a PipedReader should be used to communicate between threads. If both ends of a pipe are used by the same thread, the thread can hang. The PipedWriter class is the character-based equivalent of the byte-based PipedOutputStream.
