[Chapter 17] EventObject Chapter 17The java.util Package EventObject Name EventObject Synopsis Class Name: java.util.EventObject Superclass: java.lang.Object Immediate Subclasses: java.awt.AWTEvent Interfaces Implemented: java.io.Serializable Availability: New as of JDK 1.1 Description The EventObject class is the superclass of all other classes that represent events in the Java 1.1 event model. The class is named EventObject to avoid confusion with java.awt.Event, which was used to represent events in the old Java 1.0 event model.
