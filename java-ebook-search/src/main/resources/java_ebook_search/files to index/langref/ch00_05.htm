[Preface] Conventions Used in This Book Preface Conventions Used in This Book Italic is used for: The names of syntactic constructs and lexical structures in the Java language New terms where they are defined Pathnames, filenames, and program names Internet addresses, such as domain names and URLs Typewriter Font is used for: Anything that might appear in a Java program, including keywords, operators, data types, constants, method names, variable names, class names, and interface names Command lines and options that should be typed verbatim on the screen Tags that might appear in an HTML document Online Resources Request for Comments
