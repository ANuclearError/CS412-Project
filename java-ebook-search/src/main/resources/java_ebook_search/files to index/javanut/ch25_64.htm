[Chapter 25] 25.64 java.lang.Throwable (JDK 1.0) Chapter 25The java.lang Package 25.64 java.lang.Throwable (JDK 1.0) This is the root class of the Java exception and error hierarchy. All exceptions and errors are subclasses of Throwable.
