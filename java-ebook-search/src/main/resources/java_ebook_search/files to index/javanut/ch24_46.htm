[Chapter 24] 24.46 java.io.OutputStream (JDK 1.0) Chapter 24The java.io Package 24.46 java.io.OutputStream (JDK 1.0) This abstract class is the superclass of all output streams. It defines the basic output methods that all output stream classes provide. write() writes a single byte or an array or subarray of bytes. flush() forces any buffered output to be written. close() closes the stream and frees up any system resources associated with it. The stream may not be used once close() has been called.
