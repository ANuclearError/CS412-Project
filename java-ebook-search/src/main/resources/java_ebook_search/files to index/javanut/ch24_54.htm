[Chapter 24] 24.54 java.io.PushbackInputStream (JDK 1.0) Chapter 24The java.io Package 24.54 java.io.PushbackInputStream (JDK 1.0) This class is a FilterInputStream that implements a one-byte pushback buffer or, in Java 1.1, a pushback buffer of a specified length. The unread() methods "push" bytes back into the stream--these bytes are the first ones read by the next call to a read() method. This class is sometimes useful when writing parsers.
