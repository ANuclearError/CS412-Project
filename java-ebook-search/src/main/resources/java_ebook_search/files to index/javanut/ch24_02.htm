[Chapter 24] 24.2 java.io.BufferedOutputStream (JDK 1.0) Chapter 24The java.io Package 24.2 java.io.BufferedOutputStream (JDK 1.0) This class is a FilterOutputStream that provides output data buffering--output efficiency is increased by storing values to be written in a buffer and actually writing them out only when the buffer fills up or when the flush() method is called. Create a BufferedOutputStream by specifying the OutputStream that is to have buffering applied in the call to the constructor.
