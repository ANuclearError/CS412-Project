[Chapter 24] 24.58 java.io.SequenceInputStream (JDK 1.0) Chapter 24The java.io Package 24.58 java.io.SequenceInputStream (JDK 1.0) This class provides a way of seamlessly concatenating the data from two or more input streams. It provides an InputStream interface to a sequence of InputStream objects. Data are read from the streams in the order in which the streams are specified. When the end of one stream is reached, data are automatically read from the next stream.
