[Chapter 30] 30.19 java.util.Random (JDK 1.0) Chapter 30The java.util Package 30.19 java.util.Random (JDK 1.0) This class implements a pseudo-random number generator. nextDouble() and nextFloat() return a value between 0.0 and 1.0. nextLong() and nextInt() return long and int values distributed across the range of those data types. nextGaussian() returns pseudo-random values with a Gaussian distribution--the mean of the values is 0.0, and the standard deviation is 1.0.
