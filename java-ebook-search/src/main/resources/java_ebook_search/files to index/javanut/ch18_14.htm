[Chapter 18] 18.14 java.awt.Color (JDK 1.0) Chapter 18The java.awt Package 18.14 java.awt.Color (JDK 1.0) The Color object describes a color. The Color() constructors describe a color as red, green, and blue components between 0 and 255, or as floating-point values between 0.0 and 1.0. The class method Color.getHSBColor() creates a color using the hue/saturation/brightness color model. brighter() and darker() are useful methods to create shading effects. The getColor() methods look up a color name in the properties database and convert the resulting integer color value into a Color object.
