[Chapter 21] 21.4 java.awt.image.DirectColorModel (JDK 1.0) Chapter 21The java.awt.image Package 21.4 java.awt.image.DirectColorModel (JDK 1.0) This class is a ColorModel that extracts the red, green, blue, and alpha values directly from the bits of the pixel. The arguments to the constructor methods specify the number of significant bits in the pixel and the mask used to extract each of the color components from the pixel. The default RGB color model is a DirectColorModel.
