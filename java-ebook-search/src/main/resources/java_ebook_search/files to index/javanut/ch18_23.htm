[Chapter 18] 18.23 java.awt.FlowLayout (JDK 1.0) Chapter 18The java.awt Package 18.23 java.awt.FlowLayout (JDK 1.0) This class implements the LayoutManager interface to lay out Component objects in a Container. FlowLayout arranges components in a container like words on a page: from left to right and top to bottom. It fits as many components as it can in a row before moving on to the next row. The constructor allows you to specify one of three constants as an alignment value for the rows, and also allows you to specify horizontal spacing between components and vertical spacing between rows.
