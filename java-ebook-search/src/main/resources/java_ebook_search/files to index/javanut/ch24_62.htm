[Chapter 24] 24.62 java.io.StringBufferInputStream (JDK 1.0; Deprecated.) Chapter 24The java.io Package 24.62 java.io.StringBufferInputStream (JDK 1.0; Deprecated.) This class is a subclass of InputStream in which input bytes come from the characters of a specified String object. This class does not correctly convert the characters of a StringBuffer into bytes, and is deprecated in Java 1.1.
