[Chapter 18] 18.44 java.awt.MenuItem (JDK 1.0) Chapter 18The java.awt Package 18.44 java.awt.MenuItem (JDK 1.0) This class encapsulates a menu item with a specified textual label. A MenuItem can be added to a menu pane with the Menu.add() method. The disable() method makes an item non-selectable; you might use it to "gray-out" a menu item when the command it represents is not valid in the current context. The enable() method makes an item selectable again.
