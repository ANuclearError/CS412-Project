[Chapter 24] 24.42 java.io.ObjectOutputStream (JDK 1.1) Chapter 24The java.io Package 24.42 java.io.ObjectOutputStream (JDK 1.1) The ObjectOutputStream is used to serialize objects, arrays, and other values to a stream. The writeObject() method serializes an object or array, and various other methods are used to write primitive data values to the stream. Note that only objects that implement the Serializable interface or the Externalizable interface can be serialized. The defaultWriteObject() may only be called from the writeObject() method of a Serializable object. It allows an object to perform additional processing before or after serializing itself.
