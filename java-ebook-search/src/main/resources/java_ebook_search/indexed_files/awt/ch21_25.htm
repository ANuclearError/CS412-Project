[Chapter 21] PaintEvent Chapter 21java.awt.event Reference PaintEvent Name PaintEvent Description The PaintEvent class represents the paint and update operations that the AWT performs on components. There is no PaintListener interface, so the only way to catch these events is to override paint(Graphics) and update(Graphics) in Component. This class exists so that paint events will get serialized properly.
