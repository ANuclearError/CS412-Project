[Chapter 21] ContainerEvent Chapter 21java.awt.event Reference ContainerEvent Name ContainerEvent Description Container events are fired off when a component is added to or removed from a container. The AWT automatically deals with adding components to containers; these events are provided only for notification. COMPONENT_REMOVED public final static int COMPONENT_REMOVED Event type ID indicating that a component was removed from a container. Constructors ContainerEvent public ContainerEvent (Component source, int id, Component child) Parameters source The object that generated the event. id The event type ID of the event. child The component that was added or removed.
