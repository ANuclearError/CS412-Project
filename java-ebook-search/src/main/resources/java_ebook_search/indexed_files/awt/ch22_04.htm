[Chapter 22] DirectColorModel Chapter 22java.awt.image Reference DirectColorModel Name DirectColorModel Description The DirectColorModel class provides a ColorModel that specifies a translation between pixels and alpha, red, green, and blue component values, where the color values are embedded directly within the pixel.
