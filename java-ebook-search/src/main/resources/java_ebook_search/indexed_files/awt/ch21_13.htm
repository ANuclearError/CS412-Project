[Chapter 21] FocusListener Chapter 21java.awt.event Reference FocusListener Name FocusListener Description Objects that implement the FocusListener interface can receive FocusEvent objects. Listeners must first register themselves with objects that produce events. When events occur, they are then automatically propagated to all registered listeners.
