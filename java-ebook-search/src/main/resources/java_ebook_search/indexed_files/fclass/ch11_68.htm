[Chapter 11] WriteAbortedException Chapter 11The java.io Package WriteAbortedException Name WriteAbortedException Synopsis Class Name: java.io.WriteAbortedException Superclass: java.io.ObjectStreamException Immediate Subclasses: None Interfaces Implemented: None Availability: New as of JDK 1.1 Description A WriteAbortedException is thrown during object deserialization when the stream of data is incomplete because an exception was thrown while it was being written. Thus, WriteAbortedException represents an exception that was thrown during object serialization and serialized into the stream.
