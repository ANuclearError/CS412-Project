[Chapter 24] 24.34 java.io.LineNumberInputStream (JDK 1.0; Deprecated.) Chapter 24The java.io Package 24.34 java.io.LineNumberInputStream (JDK 1.0; Deprecated.) This class is a FilterInputStream that keeps track of the number of lines of data that have been read. getLineNumber() returns the current line number. setLineNumber() sets the line number of the current line. Subsequent lines are numbered starting from that number. This class is deprecated in Java 1.1 because it does not properly convert bytes to characters.
