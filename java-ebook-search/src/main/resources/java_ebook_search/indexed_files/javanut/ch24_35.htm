[Chapter 24] 24.35 java.io.LineNumberReader (JDK 1.1) Chapter 24The java.io Package 24.35 java.io.LineNumberReader (JDK 1.1) This class is a character input stream that keeps track of the number of lines of text that have been read from it. It supports the usual Reader methods, and also the readLine() method introduced by its superclass. In addition to these methods, you can call getLineNumber() to query the number of lines set so far. You can also call setLineNumber() to set the line number for the current line. Subsequent lines are numbered sequentially from this specified starting point.
