[Chapter 18] 18.55 java.awt.Shape (JDK 1.1) Chapter 18The java.awt Package 18.55 java.awt.Shape (JDK 1.1) This interface encapsulates some very generic information about geometric shapes. All Shape objects must have a bounding box--i.e., a rectangle that completely encloses the represented shape. When the forthcoming Java2D API is integrated with the AWT, this interface may be changed.
