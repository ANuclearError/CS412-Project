[Chapter 18] 18.16 java.awt.Container (JDK 1.0) Chapter 18The java.awt Package 18.16 java.awt.Container (JDK 1.0) This class implements a component that can contain other components. You cannot instantiate Container directly, but must use one of its subclasses, such as Panel, Frame, or Dialog.
