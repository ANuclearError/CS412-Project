[Chapter 24] 24.10 java.io.DataInput (JDK 1.0) Chapter 24The java.io Package 24.10 java.io.DataInput (JDK 1.0) This interface defines the methods required for streams that can read Java primitive data types in a machine-independent binary format. It is implemented by DataInputStream and RandomAccessFile.
