[Chapter 28] 28.25 java.net.URLStreamHandler (JDK 1.0) Chapter 28The java.net Package 28.25 java.net.URLStreamHandler (JDK 1.0) This abstract class defines the openConnection() method that creates a URLConnection for a given URL. A separate subclass of this class may be defined for various URL protocol types. A URLStreamHandler is created by a URLStreamHandlerFactory.
