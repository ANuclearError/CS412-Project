[Chapter 24] 24.7 java.io.CharArrayReader (JDK 1.1) Chapter 24The java.io Package 24.7 java.io.CharArrayReader (JDK 1.1) This class is a character input stream that uses a character array as the source of the characters it returns. You create a CharArrayReader by specifying the character array, or portion of an array, that it is to read from. CharArrayReader defines the usual Reader methods, and supports the mark() and reset() methods. Note that the character array you pass to the CharArrayReader is not copied by this class. This means that changes you make to the elements of the array after you create the input stream do affect the values read from the array.
