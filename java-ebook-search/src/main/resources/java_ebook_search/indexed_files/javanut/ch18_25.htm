[Chapter 18] 18.25 java.awt.FontMetrics (JDK 1.0) Chapter 18The java.awt Package 18.25 java.awt.FontMetrics (JDK 1.0) This class represents font metrics for a specified Font. The methods allow you to determine the overall metrics for the font (ascent, descent, etc.), and also to compute the width of strings that are to be displayed in a particular font.
