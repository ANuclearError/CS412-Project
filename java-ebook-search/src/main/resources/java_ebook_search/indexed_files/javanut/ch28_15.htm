[Chapter 28] 28.15 java.net.ServerSocket (JDK 1.0) Chapter 28The java.net Package 28.15 java.net.ServerSocket (JDK 1.0) This class is used by servers to listen for connection requests from clients. When you create a ServerSocket, you specify the port to listen on. The accept() method begins listening on that port, and blocks until a client requests a connection on that port.
