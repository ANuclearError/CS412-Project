[Chapter 18] 18.8 java.awt.Canvas (JDK 1.0) Chapter 18The java.awt Package 18.8 java.awt.Canvas (JDK 1.0) This class is a Component that does no default drawing or event handling on its own. You can subclass it to display any kind of drawing or image, and to handle any kind of user input event. Canvas inherits the event handling methods of its superclass.
