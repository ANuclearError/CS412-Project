[Chapter 21] 21.9 java.awt.image.ImageProducer (JDK 1.0) Chapter 21The java.awt.image Package 21.9 java.awt.image.ImageProducer (JDK 1.0) This interface defines the methods that any class that produces image data must define to enable communication with ImageConsumer classes. An ImageConsumer registers itself as interested in a producer's image by calling the addConsumer() method.
