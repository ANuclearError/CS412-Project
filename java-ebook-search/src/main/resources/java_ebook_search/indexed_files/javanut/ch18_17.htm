[Chapter 18] 18.17 java.awt.Cursor (JDK 1.1) Chapter 18The java.awt Package 18.17 java.awt.Cursor (JDK 1.1) This class represents a mouse cursor. It defines a number of constants, each of which specify one of the supported cursor images. Pass one of these constants to the constructor to create a cursor of the specified type, and call getType() to determine the type of an existing Cursor object. Since there are only a fixed number of available cursors, the static method getPredefinedCursor() is more efficient than the Cursor() constructor--it maintains a cache of Cursor objects that can be reused.
