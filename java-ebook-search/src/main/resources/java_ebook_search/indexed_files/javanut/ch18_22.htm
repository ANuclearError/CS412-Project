[Chapter 18] 18.22 java.awt.FileDialog (JDK 1.0) Chapter 18The java.awt Package 18.22 java.awt.FileDialog (JDK 1.0) This class represents a file selection dialog box. The constants LOAD and SAVE are values of an optional constructor argument that specifies whether the dialog should be an Open File dialog or a Save As dialog. You may specify a FilenameFilter object to control which files are displayed in the dialog. The inherited show() method pops the dialog up. For dialogs of this type, show() blocks, not returning until the user has selected a file and dismissed the dialog (which pops down automatically--you don't have to call hide()).
