[Chapter 24] 24.51 java.io.PipedWriter (JDK 1.1) Chapter 24The java.io Package 24.51 java.io.PipedWriter (JDK 1.1) PipedWriter is a character output stream that writes characters to the PipedReader character input stream to which it is connected. PipedWriter implements one half of a pipe, and is useful for communication between two threads of an application. A PipedWriter cannot be used until it is connected to a PipedReader object, which may be passed to the PipedWriter() constructor, or to the connect() method. PipedWriter inherits most of the methods of its superclass. See Writer for more information.
