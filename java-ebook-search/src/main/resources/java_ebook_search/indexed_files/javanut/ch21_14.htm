[Chapter 21] 21.14 java.awt.image.ReplicateScaleFilter (JDK 1.1) Chapter 21The java.awt.image Package 21.14 java.awt.image.ReplicateScaleFilter (JDK 1.1) This class implements an ImageFilter that scales an image to a specified pixel size. It uses a very simple scaling algorithm in which rows and columns of image pixels are duplicated or omitted as necessary to achieve the desired size. See AreaAveragingScaleFilter for a scaling filter that results in smoother images. The methods of this class are ImageConsumer methods used for communication between the image filter and the FilteredImageSource that uses it. Applications do not usually call these methods directly.
