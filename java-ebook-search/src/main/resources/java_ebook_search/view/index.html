Java eBook Search

Version 1.0

How to use

  • Use the search bar at the top to type query.
  • Hit enter or press 'select'.
  • Click results on the left to display the result..

Boolean queries

You can use boolean queries in the search bar. The queries "x AND y", "x OR y" and "NOT x" will execute the various boolean queries. Note that capitals must be used or the system will think you're searching the acutal words.
